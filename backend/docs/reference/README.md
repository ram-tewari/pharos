## Module Manifest Quick Reference

**14 Domain Modules**:
1. **annotations** - Text highlights and notes
2. **authority** - Subject authority trees
3. **collections** - Collection management
4. **curation** - Content review
5. **graph** - Knowledge graph and citations
6. **monitoring** - System health and metrics
7. **pdf_ingestion** - PDF upload and GraphRAG
8. **quality** - Quality assessment
9. **recommendations** - Hybrid recommendations
10. **resources** - Resource CRUD
11. **scholarly** - Academic metadata
12. **search** - Hybrid search
13. **taxonomy** - ML classification
14. **ingestion** - Code repository ingestion

**Planned Modules** (Phases 5-7):
- **github_integration** - Hybrid GitHub storage
- **pattern_learning** - Pattern extraction
- **context_retrieval** - LLM context assembly
